JSON Faker

Fake Online REST API for Testing and prototyping

Introduction

JSON Faker is a free online REST API that you can use whenever you need some fake data.

Example

Run this code from any console

                        
                            
fetch('https://jsonfaker.herokuapp.com/users')

                            
.then(response => response.json())

                            
.then(json => console.log(json))

Resources

/users 100 Users
/comments 400 Comments
/images 400 Images
/todos 400 Todos
Note: Resources are related with each other. Ex. Users has comments and photos. And those can be accessed via their ids.

Routes

GET /users
GET /users/:user_id
GET /comments
GET /comments?comment_id=:comment_id
GET /comments?user_id=:user_id
GET /comments?user_id=:user_id&comment_id=:comment_id
GET /images
GET /images?image_id=:image_id
GET /images?user_id=:user_id
GET /images?user_id=:user_id&image_id=:image_id
GET /todos
GET /todos?todo_id=:todo_id
GET /todos?user_id=:user_id
GET /todos?user_id=:user_id&todo_id=:todo_id
